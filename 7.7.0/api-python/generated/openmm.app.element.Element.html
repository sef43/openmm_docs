Element¶

class openmm.app.element.Element(number, name, symbol, mass)¶

An Element represents a chemical element.

The openmm.app.element module contains objects for all the standard chemical elements, such as element.hydrogen or element.carbon. You can also call the static method Element.getBySymbol() to look up the Element with a particular chemical symbol.

Element objects should be considered immutable
